Description of p3d_math.p3dMath.TMat4
A Matrix which can be either 2 by 2, 3 by 3 or 4 by 4. It can be used for transformations.

Introduction


The array m is for directly accessing the fields of the matrix. You can also use the properties for accessing columns and rows which consist of an "_" and one number for the column and another for the row. So if you want to let's say access column 0 and row 3 you use _03. The array is stored in column-major order but it only matters if you pass it to another API than OpenGL, like Direct 3D. You might need to transpose the matrix then.
