Description of p3dMath.reflect

Definition


function reflect( i, n: TVec2 ): TVec2;

Description


Will reflect an input vector using a normal vector. For a given incident vector I and surface normal N reflect returns the reflection direction calculated as I - 2.0 * dot(N, I) * N.
N should be normalized in order to achieve the desired result.
